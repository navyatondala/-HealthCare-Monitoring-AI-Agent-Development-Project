# 🏥📊HealthCare-Monitoring-AI-Agent-Development-Project
An AI-powered healthcare monitoring assistant that helps users track medications, monitor basic health metrices, and access reliable health information through an interactive dashboard. 

---
## ⭐ Features

- 💊 **Medication Tracking System** – Add and manage medicine schedules.
- 🤖 **AI Health Assistant** – Ask health-related questions and receive basic guidance.
- 📊 **Health Monitoring Dashboard** – View stored health and medication data.
- 🗄 **SQLite Database Integration** – Securely store medication and health information.
- 🖥 **User-Friendly Interface** – Built with Streamlit for simple interaction.
- 🔒 **Basic Health Data Handling** – Designed with privacy considerations.

---
## 🛠 Tech Stack

- 🐍 **Python** – Core programming language used for development.
- 🔗 **LangChain** – Framework used to build and manage the health assistant workflows.
- 🖥 **Streamlit** – Used to build an interactive and user-friendly web interface.
- 📊 **Pandas** – Handles and processes health-related data.
- 🗄 **SQLite** – Lightweight database used to store medication and health records.
- 🔧 **Git & GitHub** – Version control and project collaboration.
